|-etc
   |
   ---documentation.txt(Файл с документации)
   |
   ---quotes.json(Файл с цитатами)
   |
   ---update.txt(Файл с псведо разметкой html который описавает обновления прогграмы)
|-static
   |
   ---favicon.ico(Ярлык веб сервера) 
   |
   ---css(Папка со стилями сайта)
       |
       ---main.css(Файл с настройками html элементов(ширина,длина,цвет и тд))
|-templates(Папка с html страницами)
|-versions(Папка с предыдущими версиями прогграмы(Это сделано для лёгкого отката))
|-config.py(Файл с настройками бота(Сообщение при старте,token,пароль к базе данных mysql  и тд))
|-functions.py(Файл с функциями которые чаще всего используйться)
|-main.py(Главный файл бота в нём вся логика бота)
|-setup.py(Файл который устоновит и настроит бота)
|-web.py(Файл с логикой веб-сервера)
