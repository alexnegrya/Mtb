Устоновка MTB бота:
Приветсвую если здесь вы решили устоновить mtb бота,этот файл готов вам помочь с этим.
Итак начнём:
Windows:
1)Скачайте интерпретатор python(оффицальный сайт https://www.python.org/)
2)Устоновите интерпретатор python(Чтобы python нормально устоновился поставьте галочку возле ADD PYTHON TO PATH)
Windwos,Linux:
1)Скачайте zip архив бота
2)Распакуйте zip
3)Зайдите в распокованую папку через командную консоль
4)Запустите устоновщик бота
5)После завершения процесса устоновки скачайте MongoDb(https://www.mongodb.com/try)
ри устоновки со всем соглашайтесь и выберите тип устоновки complete
6)Создайте базу данных Telegram
7)В базе данных создайте три коллекции admins,users,chat
8)Добавте в коллекцию admins документ с полями login,password(хэшированный в md5),level
(0-максимальный уровень допуска,1-минимальный уровень допуска)
